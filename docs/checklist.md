# 签前秒检 · 核查清单（编辑部规范）

本文件是系统开发的总路线图，来源于编辑部核查规范。

## 1. 有关总书记内容 ☆☆☆
最高优先级，零容错。

## 2. 党和国家领导人
姓名、职务、排序。

## 3. 各类名称
1. **人名**含身份；外国人名翻译准确；注意人大代表、政协委员
2. **地名**：①归属关系，重点少数民族地区；②重要事件发生地；③涉历史沿革
3. **重要机构、重要会议、重要名词**（重要提法、重要概念）
4. **名人名作名篇**
5. **法律法规文件**
6. **歌曲名、歌词、影视剧名、重要节目栏目名**

## 4. 时间
1. 重要事件时间
2. 朝代、帝号纪年与公元纪年
3. 人物生卒年、名作名篇年代

## 5. 其他事实
1. **重要数据**：注意单位、核算增减比例
2. 马恩列毛邓等著作和引语
3. 法律等条文序号和内容
4. 名言警句，俗语熟语
5. 诗文内容（可涉作者、出处等）
6. 重要文化事实（历史人物、重要文物）

## 6. 图片
1. 注意黑名单作者
2. 图中能看清的文字尽量核，包括核实英文含义
3. 图文一致：位置关系、表现内容与文字说明匹配度

## 7. 图表
1. 核数值、比例
2. 核单位（公斤、亿元等）
3. 核对应：①数值/比例与正文对应 ②与实际数值对应 ③与时间对应
